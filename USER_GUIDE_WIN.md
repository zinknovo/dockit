# Dockit 法律文书自动归档工具 - Windows 使用指南

感谢使用 Dockit！这是一个专门为律师设计的法律文书自动整理工具。

## 快速开始
1. **解压文件**：将收到的压缩包解压到一个你找得到的文件夹（如 `D:\Tools\Dockit`）。
2. **运行程序**：双击运行 `dockit.exe`。
   > **注意**：由于软件未经过微软昂贵的数字签名，Windows Defender 可能会弹出“已保护你的电脑”。请点击**“更多信息” -> “仍要运行”**。
3. **初始设置**：
   - 第一次运行会弹出设置窗口。
   - **监听目录**：建议选择你的“下载”文件夹（Downloads）。
   - **归档目录**：建议选择你的案卷存放根目录（默认会在“我的文档”下创建“Dockit归档”）。
   - 点击**“保存”**。

## 核心功能
*   **自动整理**：只要你从电子送达平台下载了 PDF 传票，Dockit 会自动识别案号，并在你的归档目录下按案号创建文件夹，把文书重命名并存入。
*   **系统通知**：归档成功后，电脑右下角会弹出通知，提醒你案件的**开庭时间**。
*   **导出排期**：右键点击右下角托盘图标（深蓝色小方块），选择“导出 Excel”，即可一键生成所有案件的开庭和期限清单。

## 常见问题
*   **软件没反应？**：Dockit 运行后会缩小在右下角托盘区。
*   **无法识别图片？**：目前版本优先支持 PDF 和 Word。如果是照片，建议先转成 PDF 存入下载文件夹。
*   **网络连接失败？**：软件自带了演示用 API Key。如果网络环境极差，请在设置中检查 API 地址。
