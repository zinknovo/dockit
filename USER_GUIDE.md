# Dockit 用户使用手册 🚀

Dockit 是一款专为法律从业者设计的“法律文书自动归档”工具。它能监听你的下载文件夹，自动识别新下载的法院传票、判决书等，提取案号、案由、当事人等信息，并自动帮你重命名、归档到对应的案件文件夹。

---

### **一、 安装步骤**
1. 双击运行 `Dockit_Setup_v1.0.exe` 安装程序。
2. 安装完成后，点击桌面上的 **Dockit** 图标运行。
3. 如果看到“Dockit 已经在运行中”的提示，请查看电脑右下角的系统托盘（小箭头里）。

---

### **二、 首次使用配置**
第一次启动时，程序会弹出**设置窗口**：

1. **设置监听目录 (Watch Directory)**：
   - 建议设置为你的“下载”文件夹（Downloads）。
   - Dockit 会盯着这个文件夹，一旦有新的 PDF 进来就会处理。
2. **设置归档目录 (Archive Directory)**：
   - 设置为你存放所有案件文件的根目录（如 `D:\律师业务\2025案件`）。
   - Dockit 会在这个目录下自动创建如 `(2025)京0101民初123号_原告_被告` 这样的文件夹。
3. **登录 / 注册**：
   - 点击“登录 / 注册”按钮，注册一个账号。
   - 登录后，你的 API Token 会自动填入，点击“保存配置”。

---

### **三、 如何使用（核心流程）**
配置好后，Dockit 就在后台默默工作了：

1. **下载文书**：你从电子送达平台或邮箱下载一份 PDF 传票。
2. **自动识别**：
   - **全自动模式**：文件下载完成后，Dockit 会在 1-2 秒内自动将其重命名并移动到案件文件夹。
   - **确认模式**：Dockit 会弹窗（或在终端）显示识别到的案号，你点击“确认”后才会归档。
3. **查看结果**：去你的“归档目录”看看，文件已经按照 `案号_文书类型.pdf` 规范排好了。

---

### **四、 进阶功能**
右键点击右下角的 **Dockit 图标**，你可以：
- **查看日历**：查看近期所有案件的开庭时间。
- **导出 Excel**：一键生成本月所有案件的“开庭记录台账”和“关键期限表”。
- **设置**：随时修改模式（自动/确认）或更换目录。

---

### **常见问题**
- **没反应？** 请确认文件是 PDF 或 Word 格式，且文件名不要包含特殊字符。
- **识别错了？** 可以在“确认模式”下点击“编辑”手动修正，Dockit 会记住你的修改并变得越来越聪明。
- **如何退出？** 右键点击托盘图标 -> 退出。
